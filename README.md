# Network Scanner Tool

A Python-based network scanning tool that automates Nmap scans and generates timestamped reports.

## What it does
- Takes an IP address as input
- Automatically runs an Nmap scan
- Detects open ports and running services
- Saves a timestamped report for every scan

## Tools Used
- Python 3
- Nmap
- Kali Linux

## How to run
1. Make sure Nmap is installed
2. Run the script:
sudo python3 scanner.py
3. Enter the target IP address when prompted

## Sample Output
- Host status (up/down)
- Open ports detected
- Services running on each port
- Timestamped scan report saved automatically

## Purpose
Built as part of cybersecurity lab practice to understand network reconnaissance and port scanning concepts used in SOC analyst workflows.
